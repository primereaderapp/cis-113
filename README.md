# CIS 113 GitHub Pages Demo

This repository hosts a static GitHub Pages demo for the Net-PY interface.

## Pages entry points

- `index.html`: main landing page
- `login.html`: alternate login page
- `dashboard.html`: dashboard preview

## Publishing

GitHub Pages is configured through `.github/workflows/deploy-pages.yml`.

After the repo is pushed to the `main` branch:

1. Open the repository on GitHub.
2. Go to `Settings` > `Pages`.
3. Set `Source` to `GitHub Actions` if it is not already selected.

Your site will publish at:

`https://primereaderapp.github.io/cis-113/`
